Exercicios/Atividades sobre repetição/while feitas no Vscode utilizando Python.

1. **Validação de nota do aluno**  
   Programa que solicita a nota de um aluno e só aceita valores entre 0 e 10.

2. **Somatório dos N primeiros números inteiros**  
   Lê um número `N` e calcula a soma de todos os inteiros de 1 até `N`.

3. **Média de números até -1**  
   Lê números indefinidamente até que o usuário digite `-1`.  
   Ao final, calcula e exibe a média dos números digitados.

4. **Contagem de pares e ímpares**  
   Lê 10 números e mostra quantos são pares e quantos são ímpares.

6. **Calculadora com menu**  
   Exibe um menu com opções de soma, subtração, multiplicação e divisão.  
   O programa só encerra quando o usuário escolhe a opção `0`.  
   Inclui tratamento para divisão por zero e opções inválidas.
